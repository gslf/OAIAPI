# OAIAPI
### Unofficial OpenAI API interface in C# 
Welcome to OAIAPI, the go-to C# library for a smooth interaction with OpenAI's API. 

---

## Documentation
[READ THE DOCS](https://gslf.github.io/OAIAPI/)

## Available API
- Chat Completion 
- Audio
- Files Management 
- Fine Tuning 
- Embeddings

## API integration in progress
