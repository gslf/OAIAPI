Class Files

Promezio.OAIAPI.Capabilities.Files

The Files class provides functionalities to interact with the OpenAI REST API for file operations. It allows for uploading, listing, retrieving, deleting, and accessing the content of files.

This class is part of the Promezio.OAIAPI library and extends the Capability base class. It requires an API key and a Logger instance for initialization.
